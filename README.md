# Sistema de Manutenção de Contas

Projeto desenvolvido em C utilizando arquivos binários.

## Funcionalidades

- Cadastrar cliente em posição específica
- Consultar cliente pelo número da conta
- Atualizar saldo
- Encerrar conta
- Listar clientes
- Reiniciar leitura do arquivo usando rewind()

## Funções utilizadas

- fopen()
- fclose()
- fread()
- fwrite()
- fseek()
- rewind()

## Estrutura do registro

- Número da conta
- Nome
- Saldo
- Status (ativo/removido)

## Arquivo utilizado

clientes.dat
